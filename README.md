🔍 4PDA Update Checker
4PDA Update Checker — простое некоммерческое фанатское Android-приложение для отслеживания и сравнения версий приложений с форума 4PDA.

✨ Особенности
📊 Мониторинг версий: Автоматическое отслеживание и сравнение установленных версий с актуальными на форуме.

🤖 Умный парсинг: Автоматическое извлечение названий, ID пакетов и номеров версий из тем.

📂 Организация по папкам: Группировка приложений простым перетаскиванием (удерживайте центр элемента 1 секунду для объединения).

📱 Выбор приложения: Окно «Открыть с помощью» для выбора между браузером и официальным клиентом.

🎨 Современный интерфейс: База Material Design 3 (плавные анимации, компактный дизайн, закрытие меню касанием вне области).

🌌 AMOLED-режим: Чисто чёрная тема для экономии заряда.
